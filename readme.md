# 🎯 QA Automation Portfolio

> End-to-End testing showcase using Playwright & Page Object Model

## 🚀 Tech Stack
- **Playwright** - Modern browser automation
- **JavaScript (ES6+)** - Test scripting
- **Page Object Model** - Design pattern
- **API Seeding** - Fast and reliable for data setup

## 📂 Structure
```
tests/
├── e2e/        → Test specifications
├── pages/      → Page Object Models
└── helpers/    → Utility functions
```

## 🎯 Test Coverage (20 Scenarios)
- ✅ Authentication flows (login, register)
- ✅ CRUD operations (posts)
- ✅ Cross-browser testing (Chrome, Firefox)
- ✅ API-seeded data patterns

## 🏃 Running Tests
```bash
# Prerequisites

npm install             # Install dependencies
npx playwright install  # Install browsers

# Run Tests
npm test              # Run all tests (both browsers)
npm run test:headed   # See browser
npm run report        # View HTML report
```
