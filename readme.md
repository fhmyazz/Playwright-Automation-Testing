# 🎯 QA Automation Portfolio

> End-to-End testing showcase using **Playwright**. Combining **API Backend, Web App Frontend, and Automation Testing** into a single workflow

## 🧠 Project Overview
- **Backend API** handles authentication and data (CRUD)
- **Frontend** provide user interaction
- **Playwright** ensure system's reliability by automation testing 

## 🚀 Tech Stack
- **JavaScript (ES6+)** - Test scripting
- **Playwright** - Modern browser automation
- **Page Object Model** - Design pattern
- **API Seeding** - Fast and reliable for data setup
- **Node.js (express)** - Backend API
- **HTML/TailwindCSS** - Frontend

## 🎯 Key Features
- ✅ Authentication (Login & Register)
- ✅ Page Object Model (POM)
- ✅ Cross-browser Testing
- ✅ 40 automated test scenarios (20 tests @browsers)
- ✅ API Seeding Pattern
- ✅ API + UI hybrid testing scenarios
- ✅ localstorage Injection

## ▶️ How to Run Locally
### 1. Install Dependencies
```
npm install
cd api
npm install
cd ..
```

### 2. Run backend API
```
cd api
npm start
```

### 3. Run tests
```
npm test
```
